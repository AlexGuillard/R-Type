Detailed Description

Component that makes the entity move in a sine wave. Every value is in pixels per second. The equation is: y = amplitude * sin(frequency * (t + horizontalOffset)) + verticalOffset.

Parameters
verticalOffsetThe vertical offset of the sine wave. (start position of the entity on the y axis) (def: 0.0F)
horizontalOffsetThe horizontal offset of the sine wave. (start position of the entity on the x axis) (def: 0.0F)
speedThe speed at which the entity moves on the x axis. (def: defaultSinXSpeed = -100)
amplitudeThe amplitude of the sine wave. (def: defaultSinAmplitude = 100)
frequencyThe frequency of the sine wave. (def: defaultSinFrequency = 100)
tWhere in the sine wave the entity is. (def: 0.0F)
