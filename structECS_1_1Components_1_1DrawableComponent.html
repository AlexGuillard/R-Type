Detailed Description

Component that contains the data needed to draw a sprite.

Parameters
spriteSheetIndexThe index of the sprite sheet to use
frameRatioRatio of the frame in the texture ex: { 16, 1 } for a sprite sheet with 16 columns and 1 row
startFirst frame of the animation ex: { 3, 0 } for the 4th frame of the first row
endLast frame of the animation ex: { 8, 0 } for the 9th frame of the first row
boomerangIf true, the animation will go back and forth
fpsNumber of frames per second
frameCurrent frame
timeSinceLastFrameTime since the last frame in seconds (used internally)
forwardIf true, the animation will go forward
